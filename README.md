# Tassel

Tassel是学士帽上的流苏，这寓意着这是送给作者2027年毕业的礼物。

## 项目框架


**tassel_core**（前后端）
: 提供边缘化，预积分，动态点路标剔除，状态估计等功能

**tassel_tools**（工具模块）
: 提供数据软同步，实时显示图像、数据，读取yaml文件参数等功能

**tassel_utils**（通用模块）
: 提供断言宏定义等功能(仅头文件形式)
